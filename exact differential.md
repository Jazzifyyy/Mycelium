# Definition
The expression 
$$M(x,y)dx + N(x,y)dy$$
is called an **exact differential** in a domain $D$ if there exists a function $F$ of two real variables such that this expression equals the [[total differential of a function|total differential]] $dF(x,y)$ of $F$ for all $(x,y)\in D$.

That is, the expression is an **exact differential** in $D$ if there exists a function $F$ such that 
$$\frac{\partial F(x,y)}{\partial x} = M(x,y)$$and
$$\frac{\partial F(x,y)}{\partial y}= N(x,y)$$
for all $(x,y)\in D$.
