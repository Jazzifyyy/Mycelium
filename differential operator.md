# Definition
Let $x$ be an $n$-times differentiable function of the independent variable $t$. 

Denote the operation of differentiation with respect to $t$ by the symbol $D$ and call $D$ a differential operator. The symbol $\frac{dx}{dt}$ is denoted by $Dx$, i.e, $Dx \equiv x'$.

And similarly, 
$$D^nx = x^{(n)}$$
where $n \in \{ 1,2,\dots \}$.


Also, we write $(D+c)x$ to denote $x'+cx$ and $(aD^n + bD^m)x$ to denote $ax^{(n)} + bx^{(m)}$, where $a,b,c$ are constants.

# Example
In this notation, 
$$a_{0}x^{(n)} + \dots + a_{n}x$$
is written as
$$(a_{0}D^n + a_{1}D^{n-1}+\dots + a_{n-1}D+ a_{n})x.$$
