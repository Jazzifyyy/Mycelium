# Definition
Suppose $F$ be a function of two real variables such that $F$ has has continuous first partial derivatives in a domain $D$.

The **total differential** $dF$ of the function $F$ is defined by the formula
$$dF(x,y) = \frac{\partial F(x,y)}{\partial x}dx + \frac{\partial F(x,y)}{\partial y}dy$$
for all $(x,y) \in D$.
