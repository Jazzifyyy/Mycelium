# Definitions
## Definition 1

We shall call a function a **UC function** if it is either a function defined by one of the following:
1. $x^n$, where $n$ is a positive integer or zero,
2. $e^{ax}$, where $a \neq 0$,
3. $\sin(bx +c)$, where $b$ and $c$ are constants, $b \neq 0$,
4. $\cos(bx +c)$, where $b$ and $c$ are constants, $b \neq 0$,
5. or a function defined as a finite product of two or more functions of these four types.

## Definition 2

Consider a UC function $f$. The set of functions consisting of $f$ itself and all linearly independent UC functions that can be used to "build" (either by a constant multiple, or linear combination) all the successive derivatives of the function $f$ is called the **UC set** of $f$.
