# Statement
Suppose $f_{1}$ is a particular solution (i.e., no arbitrary constants) of
$$a_{0}(x)y^{(n)} + a_{1}(x)y^{(n-1)} + \dots + a_{n-1}(x)y' + a_{n}(x)y = F_{1}(x)$$
and suppose $f_{2}$ is a particular solution of
$$a_{0}(x)y^{(n)} + a_{1}(x)y^{(n-1)} + \dots + a_{n-1}(x)y' + a_{n}(x)y = F_{2}(x),$$
then $k_{1}f_{1}+k_{2}f_{2}$ is a particular solution of 
$$a_{0}(x)y^{(n)} + a_{1}(x)y^{(n-1)} + \dots + a_{n-1}(x)y' + a_{n}(x)y = F_{1}(x)+F_{2}(x).$$
